## Getting Started

### Create an Account

1. Go to https://eprohori.tech
2. Click "Sign Up"
3. Enter your email and password
4. Verify your email (check spam folder)
5. Start reporting threats!

### Earn XP & Tiers

Earn points by reporting verified threats:
- Report threat: **10 XP**
- Threat verified: **50 XP**
- Trend setter (first to report): **100 XP**

**Tiers**:
- 🟤 Bronze: 0-500 XP
- 🥈 Silver: 500-2000 XP
- 🥇 Gold: 2000-5000 XP
- 👑 Platinum: 5000+ XP

## Reporting Threats

### How to Report

1. Go to "Report Threat"
2. Select threat type:
   - **Phishing**: Fake login pages, password requests
   - **Scam**: Lottery wins, money requests
   - **Malware**: Suspicious downloads, virus links
   - **Fraud**: Payment fraud, identity theft
   - **Harassment**: Unwanted messages, harassment

3. Fill in details:
   - **Entity** (email, phone, URL, domain)
   - **Source** (SMS, Email, WhatsApp, etc.)
   - **Description** (what happened)
   - **Screenshots** (optional but helpful)

4. Click "Report" ✓

### Important

- ✓ Your reports help protect others
- ✓ All reports are verified by our team
- ✓ Verified reports earn you XP
- ✓ Repeat reports from same source count as one

## Understanding Alerts

### Alert Types

**🔴 Critical Alert**
- Most dangerous threats
- Sent immediately to all users in your district
- Action: Stop contact, report to authorities

**🟠 High Alert**
- Serious threats
- Sent to your district
- Action: Be cautious, don't click links

**🟡 Medium Alert**
- Moderate threats
- Sent to interested users
- Action: Stay alert, spread awareness

**🟢 Low Alert**
- Minor threats
- Visible in map but no alert sent
- Action: For your awareness

### Manage Alerts

Go to Settings → Notifications:
- Turn alerts on/off
- Choose alert types
- Select delivery method (SMS, Email, App)

## Using the Chatbot

### What is the Chatbot?

EProhori's AI chatbot analyzes threats and provides guidance. Available 24/7.

### How to Use

1. Click "Chat with EProhori"
2. Describe your threat:
   - "I received a phishing SMS asking for my password"
   - "Is this email safe: [sender]?"
   - "Someone asked me to send money online"

3. Get instant analysis with:
   - **Threat Type** (what it is)
   - **Severity** (how dangerous)
   - **Action Steps** (what to do)
   - **Prevention Tips** (how to avoid)

### Example Conversations

**User**: "I got an email saying my account will be locked"
**Chatbot**: "This is likely phishing. Banks never ask for passwords via email. Delete it and ignore."

**User**: "A friend won a lottery?"
**Chatbot**: "Classic scam. No one wins random lotteries. Don't send money. Warn your friend."

**User**: "Is bit.ly/xyz123 safe?"
**Chatbot**: "URL shorteners are risky - can hide malware. Avoid clicking unless you trust the source."

## Web Extension

### Install

1. Go to Chrome Web Store (coming soon)
2. Click "Add to Chrome"
3. Click "Add Extension"
4. Pin to toolbar for easy access

### Features

✓ **Real-time Blocking**: Warns before visiting dangerous sites
✓ **Blocklist Updates**: Automatic threat database updates
✓ **One-click Reporting**: Report suspicious sites directly
✓ **Safe Browsing**: Green checkmark for safe sites

### How It Works

1. **Before you click a link**: Extension checks if it's dangerous
2. **Suspicious site?**: Warning popup appears
3. **Want to report?**: Click "Report to EProhori"
4. **Database updates**: New threats added within minutes

## Account Management

### Profile Settings

1. Go to "My Account"
2. Update:
   - Profile picture
   - Display name
   - Bio
   - Preferred language (Bengali/English)

### Privacy Settings

- **Data visible to**: You choose (Public/Private/Friends only)
- **Report anonymity**: Report as "Anonymous" if desired
- **Email notifications**: Control when you receive emails

### Security

- **Change password**: Settings → Security → Change Password
- **2FA (Two-Factor Auth)**: Recommended for extra security
  1. Go to Settings → Security → Enable 2FA
  2. Scan QR code with Google Authenticator
  3. Enter 6-digit code to confirm

### Delete Account

Settings → Account → Delete Account (irreversible!)

## FAQ

### General Questions

**Q: Is EProhori free?**
A: Yes! Completely free. No ads, no premium tiers.

**Q: Is my data safe?**
A: Yes. We use encryption, secure servers, and don't sell your data.

**Q: How do I contact support?**
A: Email: support@eprohori.tech or use the in-app chat.

**Q: What languages does EProhori support?**
A: Bengali and English. More languages coming soon.

### Reporting Questions

**Q: What if my report is rejected?**
A: We verify all reports. If rejected, it may need more evidence. You can resubmit with additional details.

**Q: How long does verification take?**
A: Usually 24-48 hours. Urgent threats faster.

**Q: Can I report anonymously?**
A: Yes! Uncheck "Public report" when reporting.

**Q: What's a verified threat?**
A: Our team confirmed it's actually dangerous. Verified reports earn you XP!

### Technical Questions

**Q: Why didn't I get an alert?**
A: Check Settings → Notifications. May be turned off or filtered.

**Q: Can I export my data?**
A: Yes! Account → Download My Data (coming soon)

**Q: Does the extension work on all sites?**
A: Works on all websites except Chrome Web Store itself.

**Q: How often is the blocklist updated?**
A: Every 30 minutes automatically.

### Security Questions

**Q: Can hackers see my reports?**
A: No. Reports are encrypted and protected.

**Q: Is 2FA necessary?**
A: Highly recommended, especially if you report threats.

**Q: What if I get hacked?**
A: Change password immediately. Contact support. We'll help reset your account.
